# Stock Harvest AI

人間には不可能な東証全銘柄（約3,800社）の24時間監視をAIが実行し、投資チャンスを逃さないシステム

## 🎯 特徴

- **全銘柄AIスキャン**: 3,800銘柄を5分以内で分析
- **自動価格算出**: エントリー・利確・損切り価格を自動計算
- **リアルタイム通知**: LINE連携で重要なシグナルを即座にお知らせ
- **美しいUI**: 青・緑・白を基調とした見やすいデザイン

## 🚀 デモサイト

[こちらからアクセス](https://stock-harvest-ai.vercel.app) ※準備中

## 📱 主な機能

1. **AIダッシュボード** - 注目銘柄をワンタッチで確認
2. **銘柄詳細分析** - テクニカル・ファンダメンタル両面から分析
3. **ポートフォリオ管理** - 保有銘柄の損益を一元管理
4. **アラート設定** - 価格到達時に自動通知
5. **バックテスト** - 過去データで戦略を検証

## 🛠 技術スタック

- Frontend: React 18 + TypeScript + MUI
- Backend: Python + FastAPI
- Database: PostgreSQL (Neon)
- Deployment: Vercel + Google Cloud Run
